# 📊 Expense Tracker - Personal Finance Manager

Maine yeh project apni finance tracking problems ko solve karne aur JavaScript ki fundamentals (DOM aur Local Storage) ko samajhne ke liye banaya hai. Yeh ek simple, fast aur responsive web app hai jo bina kisi server ya database ke kaam karti hai.

## ✨ Main Features

* **Quick Dashboard:** Ek hi jagah par balance, total income aur kharche dekh sakte ho.
* **Data Persistence:** Isme maine `localStorage` use kiya hai, toh browser band karne ke baad bhi aapka data delete nahi hoga.
* **Custom Categories:** Food, Travel, Bills jaise categories mein kharche divide karne ka option.
* **Statement Download:** Aap apne poore mahine ka record `.csv` ya `.txt` format mein download kar sakte ho (JavaScript Blob use karke).
* **Fully Responsive:** Bootstrap use kiya hai taaki mobile aur laptop dono par perfect dikhe.

---

## 🛠️ Tools & Tech Used

* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Logic:** Vanilla JavaScript (ES6)
* **Storage:** Browser LocalStorage (No Backend required)
* **Export Feature:** JavaScript Blob API

---

## 📂 File Structure

* `index.html` - App ka main structure.
* `style.css` - Custom design aur look.
* `script.js` - Saari functionality aur calculations.
* `storage.js` - Data save aur delete karne ka logic.
